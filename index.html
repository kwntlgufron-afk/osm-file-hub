⚙ GitHub Setup

Connect your GitHub account to store and serve files via raw.githubusercontent.com — the backbone of OSM-compatible hosting.

Why GitHub? Files stored in public GitHub repos are accessible via a permanent, CDN-like URL:
https://raw.githubusercontent.com/{user}/{repo}/{branch}/{path}
This URL is fully compatible with OpenStreetMap tile/data servers.

Steps to get your token:

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click Generate new token (classic)
  3. Set expiration to No expiration (or any period)
  4. Check repo scope (full control of private repositories)
  5. Click Generate token and copy it
